# snip 🦀

**snip** is a lightning-fast, local-first snippet manager for terminal power users. Stop digging through `.txt` files or old projects—save your best one-liners and complex boilerplate in a fuzzy-searchable database that lives in your terminal.

> *snip — High-performance, fuzzy-searchable, and zero-lag. Just your code when you need it.*

## ✨ Features

- ⚡ **Instant Search**: Powered by the `nucleo` fuzzy-match engine.
- 📂 **Smart Organization**: Categorize with tags and add rich descriptions.
- 🏠 **Privacy First**: Everything is stored locally in your OS's standard data directory.
- 🛡️ **Robust**: Built with Rust for memory safety and blazing speed.
- 💻 **Clean UI**: Visual grouping by tags and formatted output.
